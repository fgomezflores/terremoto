Trabajo Unidad 2: visualización de datos con D3

Análisis de datos sobre un terremoto en Twitter

Francisco Jesús Gómez Flores (fgflores@uma.es)

Introducción

El presente trabajo se enmarca dentro de la Unidad 2 de la asignatura “Informática Social: tecnologías y métodos” donde se completa el proyecto de visualización que se ha estado llevando a cabo durante esta segunda parte de la asignatura.

El ejercicio se centra en la visualización de datos con D3. Mediante este framework de desarrollo se realiza una visualización en profundidad de una visión de los datos que se han trabajado durante el curso.

En este sentido, se utilizará el dataset tweets que contienen los hashtags #Lindol y #EarthquakePH del 27 de julio al 3 de septiembre de 2022 obtenido del repositorio Kaggel.

Adicionalmente, también se usará una fuente de información oficial que tiene el registro de terremotos producidos entre el 27 de julio al 3 de septiembre de 2022. En concreto del Servicio Geológico de Estados Unidos​​ o USGS.

Tecnología utilizada

Para el desarrollo de este trabajo se utiliza la siguiente tecnología:

  • Bootstrap para la construcción del espacio web.
  • D3 para la visualización de datos, en concreto, se utiliza D3.js graph gallery
  • Para los análisis previos de datos se ha utilizado, además de Oracle SQL Developer (ya utilizada en anteriores ejercicios), las herramientas iNZight y Oracle Analitics Desktop.
  • Asimismo, se han utilizado para la explotación de datos Python y las herramientas Microsoft Access y Excel.
